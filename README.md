##  Problem Statement

Students often struggle to manage multiple internship and placement applications, which can lead to confusion, errors, and missed opportunities.

Administrators need a system that's easy to use and straightforward. This system should help them look at a lot of applications keep track of them and manage them without having to use spreadsheets or do everything by hand. Administrators want to be able to review applications and stay on top of things. Administrators need something that will make their job easier when it comes to handling volumes of applications.



##  Solution

Coding Protocol provides a system that helps manage applications, in one place, which's the **centralized application management system** that does a few things like:

- Enables students to submit applications through a clean and user-friendly interface

- Allows administrators to securely view and manage all applications in one place

- Stores and updates data in real time using cloud-based services

This makes things better for students and administrators. It helps them do things efficiently and they can see what is going on. The user experience is also better, for both students and administrators.


##  Student Features

- Apply for internships by submitting complete personal and academic details

- Clean and intuitive user interface

- Real-time submission and storage of applications

- Resume upload and eligibility-based application flow


##  Admin Features

- Secure admin login system

- View all student applications in a structured table format

- Approve or reject applications with a single click

- Clear all applications when required

- Application status tracking (Pending, Approved, Rejected)
